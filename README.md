### 如何安裝：
* 對於  `*.Lua`  脚本文件：在 Aseprite 上依次打開 ***<ins>文件/脚本/打開脚本文件夾</ins>*** 然後把這些脚本文件放入到此文件夾中。  
* 對於  `*.aseprite-extension`  脚本文件：在 Aseprite 上依次打開 ***<ins>編輯/首選項/拓展</ins>*** 然後安裝這些脚本文件。
